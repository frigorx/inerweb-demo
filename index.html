Vision

Un socle unique pour tout le campus

Architecture pensée pour être déployée sur toutes les filières sans surcoût

🏗️

Socle générique et réutilisable

Chaque nouvelle filière bénéficie de l'infrastructure existante : hébergement, backend, authentification. Le même moteur, adapté à chaque référentiel.

💰

Coût : 0 € pour l'établissement

GitHub Pages (hébergement gratuit) + Google Apps Script (backend gratuit) + Google Sheets (base de données gratuite). Aucun serveur, aucune licence.

⚙️

Architecture légère et pérenne

HTML / CSS / JavaScript pur en frontend. Google Apps Script en backend. Google Sheets en base de données. Pas de framework, pas de dépendances, pas d'obsolescence.

🚀

Déployable sur 30 UFA

Le campus ÉQUATIO dispose de 30 UFA dédiés à l'apprentissage. Chacun peut bénéficier d'outils inerWeb adaptés à ses besoins spécifiques.

👨‍🏫

Conçu par un enseignant

F. Henninot, enseignant au lycée Jacques Raynaud, a développé bénévolement cet écosystème complet. Un outil pensé par et pour le terrain.

📱

100% responsive et accessible

Toutes les applications fonctionnent sur smartphone, tablette et ordinateur. Aucune installation requise, un simple navigateur suffit.
