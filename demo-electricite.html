Formules essentielles
Puissance monophasee
P = U x I x cos phi
P en W, U en V, I en A
Puissance triphasee
P = U x I x racine(3) x cos phi
U = 400V entre phases
Courant monophase
I = P / (U x cos phi)
Courant triphase
I = P / (U x racine(3) x cos phi)
Chute de tension (%)
dU% = 100 x 2L x I x (rho/S x cos phi + lambda x sin phi) / U
rho cuivre = 0.0225 ohm.mm2/m | lambda = 0.08 mOhm/m
Loi d'Ohm
U = R x I
Puissance electrique
P = U x I
